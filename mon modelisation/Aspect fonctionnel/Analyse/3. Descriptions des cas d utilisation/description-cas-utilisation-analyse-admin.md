# Descriptions — ADMIN

- **Login** : authentification par email/mot de passe, blocage IP et statut actif requis.
- **CRUD clients** : creation/modification/suppression fiches client et contact.
- **CRUD locations** : creation d'un bail (dates coherentes, unite associee) et suivi statut.
- **Gerer documents** : upload pieces, typage obligatoire selon scenario.
- **Payer abonnement** : choix methode (mobile, cash), regles de mois requis, gestion doublons.
- **Importer clients** : import CSV, validation par ligne, stockage des erreurs.
- **Undo personnels** : rollback d'actions autorisees sur une fenetre courte.
