# SOP Operations - Backend KYA

Ce dossier contient des SOP (Standard Operating Procedures) en format court,
concu pour etre imprime et execute pas a pas.

Mode d'utilisation:
1. Identifier le scenario.
2. Ouvrir la SOP correspondante.
3. Suivre les etapes dans l'ordre.
4. Documenter la preuve de traitement.
5. Escalader si un critere de blocage est present.

Liste des SOP:
1. `SOP-01-Validation-Demande-Admin.md`
2. `SOP-02-Deblocage-IP.md`
3. `SOP-03-Regularisation-Abonnement-Admin.md`
4. `SOP-04-Validation-Paiement-Especes-SuperAdmin.md`
5. `SOP-05-Incident-Paiement-Provider.md`
6. `SOP-06-Echec-Import-Clients.md`
7. `SOP-07-Rollback-Action.md`
8. `SOP-08-Triage-Alertes-Securite.md`

Rythme recommande:
- Quotidien: SOP-01, SOP-08
- Hebdomadaire: SOP-02, SOP-06
- A l'incident: SOP-03, SOP-04, SOP-05, SOP-07
